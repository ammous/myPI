# Plugin Domoticz For Yana-Server


http://forum.idleman.fr/discussion/251/yana-domoticz/p1

## Features
 * Configuration de la connexion au serveur Domoticz
 * Affichage de l'ensemble des commandes vocales disponible pour le serveur Domoticz (switch,Scenes/Groups,Temp,UserVariables)
 * Activation/désactivation des commandes Vocales Domoticz
